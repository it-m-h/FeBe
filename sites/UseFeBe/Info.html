Ordnerstruktur

Ordner Info

 Backend - Applikation - Hier werden die Model und Controller abgelegt.

 Hier werden die Rounting und Konfigurationsdateien abgelegt.

PHP
router.php

 Hier werden die Datenbanken und Logdateien abgelegt.

error
 Hier werden die Fehlerseiten abgelegt.
lib
 Hier werden die Bibliotheken abgelegt.
public
 Hier werden die öffentlichen Dateien abgelegt.
sites
 Hier werden die Frontend - Applikationen abgelegt. (Views)
templates
 Hier werden die Handlebars - Templates und Partials abgelegt.

hbs
index.hbs

Seiten erstellen

  1. Seite erstellen in sites (.php, html oder md)
  2. Seite in navigation eintragen(templates/partials/Nav.hbs) (<li><a class="" href="/XXX">XXX</a></li>)

API - Schnittstelle

  1. Klassen (Model und Controller) erstellen in app

Datenbank

  • Standardmässig wird eine SQLite Datenbank verwendet
  • Die Datenbank wird in data abgelegt
  • Die Datenbank kann in der lib/database.php verändert werden ($this->db = new \PDO('sqlite:'.BASEPATH.'data/FeBe.sqlite3');)

Tipp: es kann auch eine MySQL Datenbank oder sonst eine verwendet werden.
